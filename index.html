📋Tunnel Type Formulas
🔵 Underlay (per peer pair)
CC group N×M + exact-match for cc=none labels. Hub↔Spoke + Hub↔Hub: all labels. Spoke↔Spoke: Any-topo labels only. Gated by BIO label union — only forms when at least one overlay would use the labels.
🟠 Overlay (per peer pair)
1 per applicable BIO. Hub-Spoke BIOs: hub↔spoke + hub↔hub. Mesh BIOs: additionally spoke↔spoke. BIO must have ≥1 matching label on both sides to form.
🟢 Passthrough (local, per device)
Interface anchor (1 per WAN label) + BIO×Label (label_count × BIO_count). LAN-side passthroughs excluded (site-specific). SSE adds 2×labels×pairs.
